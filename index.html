Declarative YAML specifications for agent definition. Links to architectural decisions, compiles to WASM, and enables version-controlled agent behavior.

Production protocol implementations including MCP (v0.2.3), authentication systems, and LLM provider abstraction for Ollama, OpenAI, and Anthropic.

airssys Available
Actor runtime with supervision trees (v0.1.0), WASM component engine, and OS abstraction layer. Benchmarked at ~625ns actor spawn time and 4.7M msgs/sec throughput.
Repository Documentation
airsdsp In Development
Rust implementation of DSP framework (arXiv:2212.14024). Provides explicit control over language model and retrieval model composition in multi-stage pipelines.
Repository Documentation
airsdlc Framework
Implementation of AWS AI-DLC methodology. Structured workflow from requirements (PRD) through domain modeling (DAA) to architecture decisions (ADR) and code deployment (Bolts).
